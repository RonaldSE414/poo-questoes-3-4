# Lista de POO - Questões 3 e 4

Este projeto contém a resolução das questões 3 e 4 da lista de Programação Orientada a Objetos.

## Organização

O projeto está dividido em duas pastas:

- questao3 → Sistema de ingressos usando herança
- questao4 → Sistema de imóveis com acréscimo e desconto

## Questão 3

Implementação das classes:
- Ingresso
- VIP
- Normal
- CamaroteInferior
- CamaroteSuperior

Utiliza herança para especializar os tipos de ingresso.

## Questão 4

Implementação das classes:
- Imovel
- Novo
- Velho

Permite calcular o valor final com acréscimo ou desconto.

## Tecnologias
- Java
- Programação Orientada a Objetos
